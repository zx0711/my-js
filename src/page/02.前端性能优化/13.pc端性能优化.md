网络加载

- 减少 http 请求次数
- 减少 http 请求大小
- 将 css 和 js 放在外部文件中，避免使用 style 和 script 标签引入
- 避免页面中空的 href 和 src,因为就算为空，浏览器也会继续加载的
- 减少页面重定向
- 最好使用 get 请求
