#### 1,web app

- web 前端工程师学的就是这个
- 页面请求 url,url 在服务器上，肯定需要网络
- 开发迭代快，成本低，比如修改一个样式比较方便

#### 2,native app（原生开发）

- 安卓和 ios 开发，安卓用户下载一个 apk 文件，到手机本地，所以打开页面是可以不需要网络，但是数据在服务端，它是原生开发，所以很快，很流畅，但是迭代不方便

#### 3,混合 app

- web app 和 混合 app 结合，既有安卓，ios.又有 web app 页面
